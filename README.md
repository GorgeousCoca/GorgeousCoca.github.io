# Zanini Restaurants — MVP сайт

Статический прототип сайта ресторанной группы Zanini Restaurants.

## Структура проекта

- `index.html` — главная страница (о компании, рестораны, афиша, банкетный блок, кондитерская, карьера, форма бронирования).
- `little-italy.html` — страница ресторана Little Italy (история, Сальваторе, меню, команда, события, карта, бронь).
- `career.html` — раздел «Карьера» с преимуществами, вакансиями, формой отклика и ценностями.
- `news.html` — «Новости и афиша» с лентой событий и новостей.
- `banquets.html` — страница «Банкеты и кейтеринг» с формой запроса.
- `contacts.html` — контакты, карта всех ресторанов и форма обратной связи.
- `styles.css` — общие стили (типографика, сетки, карточки, формы, адаптив).
- `main.js` — лёгкая JS-логика (демо-слоты бронирования, меню, вакансии, новости, обработка форм).
- `assets/` — статические ресурсы:
  - `assets/images/` — фотографии ресторанов, блюд, команды (для подстановки вместо плейсхолдеров).
  - `assets/icons/` — иконки интерфейса.
  - `assets/fonts/` — кастомные шрифты (если понадобятся).
- `data/` — заготовка под JSON‑данные (меню, новости, вакансии) для будущей интеграции с CMS/бэкендом.

Проект рассчитан на статический хостинг; интеграцию онлайн‑бронирования и онлайн‑кассы можно добавить позднее через внешние виджеты или лёгкий бэкенд.
