Разработка backend-сервиса таск-трекера

Почти с нуля создал backend-сервис для таск-трекера на Spring Boot. По сути, заставил джавовское приложение общаться с миром через REST API и дружить с базой данных. Чтобы заказчик мог легко всё потестировать, упаковал это в Docker-контейнер и даже закинул в Kubernetes (пришлось гуглить, но всё завелось!). В итоге, развернуть тестовую среду стало можно за пару минут, а не часов.
